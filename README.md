# youtube-checker

To use the python file "getTranscript.py", create and activate a conda environment with the following packages:
    conda install pip
    pip install serpapi
    pip install google-search-results
    pip install -q -U google-genai

Then, create an account with serpapi and place your api key in the .env file
    https://serpapi.com/dashboard
Do the same with google ai studio:
    https://aistudio.google.com/
